<details>

  ### ⚙️  Architecture

  This profile uses **two** GitHub Actions workflows to keep everything dynamic:

  | Workflow | File | Runs | Generates |
  |---|---|---|---|
  | **Metrics** | `.github/workflows/metrics.yml` | Daily + on push | Full dashboard, isocalendar, habits, languages, achievements, stars, LOC |
  | **Snake** | `.github/workflows/snake.yml` | Daily | Contribution grid snake (SVG + GIF) |

  ### 🔧  Setup

  1. Fork this repo and rename it to `Nithin-3/Nithin-3`
  2. Create a fine-grained PAT with `repo` and `user` scopes
  3. Add it as `METRICS_TOKEN` in **Settings → Secrets and variables → Actions**
  4. Push to `main` — workflows trigger automatically

  ### 📦  Outputs

  All generated assets live under [`assets/`](https://github.com/Nithin-3/Nithin-3/tree/main/assets):

  ```
  assets/
  ```
